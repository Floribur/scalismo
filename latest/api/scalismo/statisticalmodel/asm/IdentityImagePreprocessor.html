Companion object IdentityImagePreprocessor

case class IdentityImagePreprocessor(ioMetadata: IOMetadata = IdentityImagePreprocessor.IOMetadata_Default) extends ImagePreprocessor with Product with Serializable

The "identity" Preprocessor performs no preprocessing. In other words, this class can be considered an adapter that turns a discrete image into a PreprocessedImage, without modifying the image.
